# 组合模式的“部分-整体”层次结构

**菜单项（Leaf）**：单个菜单项。

**子菜单（Composite）**：可以包含多个菜单项或子菜单。

**操作**：可以统一地显示菜单或子菜单内容，客户端无需区分。

```java
interface MenuComponent {
    void display();
}

class MenuItem implements MenuComponent {
    private String name;

    public MenuItem(String name) {
        this.name = name;
    }

    @Override
    public void display() {
        System.out.println("MenuItem: " + name);
    }
}

class SubMenu implements MenuComponent {
    private String name;
    private List<MenuComponent> components = new ArrayList<>();

    public SubMenu(String name) {
        this.name = name;
    }

    public void add(MenuComponent component) {
        components.add(component);
    }

    public void remove(MenuComponent component) {
        components.remove(component);
    }

    @Override
    public void display() {
        System.out.println("SubMenu: " + name);
        for (MenuComponent component : components) {
            component.display();
        }
    }
}

public class Main {
    public static void main(String[] args) {
        MenuItem item1 = new MenuItem("File");
        MenuItem item2 = new MenuItem("Edit");

        SubMenu subMenu = new SubMenu("Help");
        subMenu.add(new MenuItem("About"));
        subMenu.add(new MenuItem("FAQ"));

        SubMenu mainMenu = new SubMenu("Main Menu");
        mainMenu.add(item1);
        mainMenu.add(item2);
        mainMenu.add(subMenu);

        mainMenu.display();
    }
}

```
